RateIt plugin

Prerequisites

In order to use RateIt you'll need:

  • jQuery 1.4.3 or newer.
  • jquery.rateit.min.js, rateit.css, delete.gif, star.gif
  • Place a reference to the rateit.css stylesheet into the page head.
  • Place a script reference to jquery.rateit.js before your </body> tag.

Options

Examples

  • 1) Basic usage

                <div class="rateit">
            </div>

  • 2) Progressive enhancement 

                <input type="range" min="0" max="5" value="0" step="0.5" id="backing2">
            <div class="rateit" data-rateit-backingfld="#backing2">

  • 3) Readonly and preset value

                    <div class="rateit" data-rateit-value="2.5" data-rateit-readonly="true"></div>

  • 4) More settings

                <input type="range" min="0" max="10" value="0" step="0.25" id="backing4">
            <div class="rateit" data-rateit-backingfld="#backing4" data-rateit-resetable="false"
                data-rateit-step="0.25" data-rateit-ld="0" data-rateit-ub="10">
            </div>

  • 5) Javascript interaction

                <div class="rateit" id="rateit5" data-rateit-min="2">
            </div>

            <script type="text/javascript">
                $("#rateit5").bind('rated', function () { alert('You\'ve rated it: ' + $(this).data('rateit-value')); });
                $("#rateit5").bind('reset', function () { alert('Rating reset'); }); 
            </script>

  • 6) Javascript invocation

                <input type="hidden" id="backing6">
            <div id="rateit6">
            </div>

            <script type="text/javascript">
                $(function () { $('#rateit6').rateit({ max: 20, step: 2, backingfld: '#backing6' }); });
            </script>

  • RTL support

    Now we are in RTL mode. No changes needed to make this work.
